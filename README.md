# Web Scraping Automation Tool

A Python-based web scraping automation tool using Selenium that extracts data from websites and exports to CSV.

## Features
✅ Automated browser control with Selenium
✅ Multi-page web scraping
✅ Data cleaning with pandas
✅ CSV export
✅ Error handling and logging

## Installation

```bash
pip3 install -r requirements.txt
```

## Usage

### Simple Version (Beginner)
```bash
python3 simple_scraper.py
```

### Advanced Version (Production)
```bash
python3 web_scraper_automation.py
```

## Output
- `quotes.csv` - Simple version output
- `quotes_data.csv` - Advanced version output

## Technologies
- Python 3.9+
- Selenium 4.0+
- Pandas
- WebDriver Manager
